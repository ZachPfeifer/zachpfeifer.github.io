Highland Authority

Vue.js / Node.js (MVC Pattern)

Highland Authority is a stat tracker website for Athletes that participate in Highland Games Events. On this site you can view all paricpants scores and contact info. This website also support the Judges for the games. Judges can add Athletes stats live while game is being played. Judges can also create Games for Athletes to compete in.

Inspire

JavaScript / Node.js (MVC Pattern)

Inspire is an App ment to be used as a home page for your browser that allows you to see a random image as a background with the time being displayed as well as your local weather. Inside the Inpire page you can also make a To Do list for tasks yopu have to do for the day.

Boss Fight

HTML / JavaScript

Boss Fight is a basic browser game where you must use items or actions to kill the boss. You have 3 different characters that can do actions to reduce the Bosses health.

Total Rick'All

C# (MVC Pattern)

A C# Console App inspired by the Console Adventure game Zork. In this game you move room to room to find parsites to kill before you can win the game.

Kanban Board

Vue.js / Node.js (MVC Pattern)

Kanban Board is a website inspired by trello boards. In this app you are able to make an account then create boards that can have tasks and comments appended to them.
